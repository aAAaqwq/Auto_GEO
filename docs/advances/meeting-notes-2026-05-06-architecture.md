## 二、PostgreSQL 迁移方案（CTO）

4. 现有 SQLite 数据 — 编写一次性迁移脚本

**信心度**: 90%

## 四、合规风险（CRO）

**调研结论：**
- 知乎/百家号/CSDN 均无官方发布 API
- CDP/Playwright 本身合法，但违反平台 ToS 存在封号风险
- 目前国内无相关刑事判例
- **建议**: 产品定位为"辅助工具"而非"全自动爬虫"，用户操作+半自动模式降低风险

## 六、部署建议（CTO + CRO）

**推荐混合模式：**
- **云端（SaaS）**: 数据分析 + 内容生成 + 关键词分析
- **本地（Agent）**: 浏览器自动化发布

**部署架构：**
```
阿里云轻量VPS
├── Nginx (反向代理 + SSL)
├── Docker Compose
│   ├── FastAPI 后端
│   ├── PostgreSQL
│   └── Redis (可选，缓存/队列)
└── Let's Encrypt (自动SSL)
```

**CI/CD**: 利用现有 GitHub Actions 改造
**预估工时**: 7小时
